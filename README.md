# Nexion Chat SDK

Integrate secure, encrypted chat into your crypto app in under 5 minutes.

## Installation

```bash
npm install nexion-chat-sdk
```

## Quick Start

```javascript
import { NexionChat } from 'nexion-chat-sdk';

// Inside your component
<NexionChat 
  apiKey="YOUR_LICENSE_KEY"
  theme="dark" 
  userId="user_123"
/>
```
